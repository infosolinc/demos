# SAP BusinessObjects Version Management with GitHub

This collection brings together official SAP documentation and community guidance for configuring and using **GitHub-based version management** with **SAP BusinessObjects BI Platform**.

## Official SAP Resources

- **SAP Community Blog**  
  *How to use Version Management with GitHub Repository*  
  A step-by-step walkthrough explaining how GitHub version management integrates with the BI Platform, including configuration concepts and practical usage.  
  https://community.sap.com/t5/technology-blog-posts-by-sap/how-to-use-version-management-with-github-repository/ba-p/13811133

- **SAP Note 3335429**  
  *Configure Git / GitHub for version management in BI Platform 4.x*  
  The authoritative SAP Note detailing supported configurations, prerequisites, limitations, and troubleshooting guidance for GitHub-based version control.  
  https://me.sap.com/notes/3335429/E

## Recommended Reading Order

1. Start with **SAP Note 3335429** to understand supported versions, requirements, and constraints.
2. Follow with the **SAP Community blog** for practical examples and usage patterns.

## Notes

- Always reference the SAP Note for the most current support status and corrections.
- SAP Community content reflects practical experience but may lag official support updates.
